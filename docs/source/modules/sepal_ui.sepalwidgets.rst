sepal\_ui.sepalwidgets package

.. tip::

   All the submodule method and class are wrapped in the package :code:`__init__.py` file.
   They can thus be called from the :code:`sepal_ui.sepalwidgets` package

      from sepal_ui import sepalwidgets as sw

      date_picker = sw.DatePicker()
