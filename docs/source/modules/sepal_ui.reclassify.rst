sepal\_ui.reclassify package
============================

.. tip::

   All the submodule method and class are wrapped in the package :code:`__init__.py` file.
   They can thus be called from the :code:`sepal_ui.reclassify` package

   example : 
   
   .. jupyter-execute:: 

      from sepal_ui import reclassify as rec
      import ipyvuetify as v
      
      v.theme.dark = False
    
      rec.ReclassifyTile(gee=False)
